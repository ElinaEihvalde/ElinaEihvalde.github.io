#SLOVAKFOLKLORE

Fully responsive website prototype for online community designed and developed in collaboration with 3 students.

UX design / UI design / Content / Development

Overview

The focus was to create an official website for an online community that has gained more than 10k followers in the last year. As Slovakfolklore was on several social media platforms they needed one platform that would increase the Association's credibility, broaden the community and showcase their ideas and values as well as inform the community about the latest Slovak folklore events all around the world.

TIME

4 weeks

ROLE

Research, UX/UI designer, User testing, Front-end developmer

CASE

Second semester exam project built without any frameworks or libraries

Step 01

Research the problem

We started by understanding how #SLOVAKFOLKORE works what are the needs and who are the users. At this point, I created a project and time plan that was followed throughout the project.

Step 03

Interviews

We conducted interviews where we learned about user personal relation with their country's traditions, how they understand globalization and their opinion on the combination of the traditions with the modern culture.

Step 05

Mockups

We created hi-fi visual and prototypes with some functionality in Adobe Xd. We worked with semiotics, color theory and tried working with Material design. This was the final design stage before starting the development process.

Step 02

Survey

To understand what #SLOVAKFOLKORE's users are most interested in, we created a  survey which that was posted on both of #SLOVAKFOLKORE’s social media accounts

Step 04

Sketches and wireframes

The design process started with brainstorming to create different layout options that were based on research. This phase was stating point for prototyping where we gained feedback and refined the product.

Step 06

Front-end development

For this project I was responsible for the development process. We created a responsive website by using HTML, CSS, JavaScript and jQuery. As the requirements forbid to use any 3rd party code it was challenging but I learned a lot about working with JavaScript had to learn a lot to implement functionality.

Final prototype

en developing this website we tried to combine modern UI elements with a traditional theme, moreover we worked with semiotics and created adaptions of Slovak national colors. We created a prototype for a platform where the organization's followers could learn about upcoming events, read about different folk groups and contact them. To achieve this we created a simple filtration function so users could separate the ensembles by region, alphabetically and by type, but events by type and month.
