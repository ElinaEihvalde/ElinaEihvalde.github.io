Aalborg Karneval

Web app prototype for the biggest carnival in Northen Europe. Project developed in a group of 4 students.

User journey / IA / UI design / Development

Overview

Aalborg Karneval is the biggest carnival in Northen Europe, however, they don't have a platform that provides the participants with valuable information on the go. This web app prototype was designed and developed as a possible solution to help participants navigate, find nearest facilities, restaurants and more.

TIME

3 weeks

ROLE

UX/UI designer, Front-end development

CASE

Create web app that is relevant to user groups needs

Step 01

Research the problem

We started the project by researching the carnival and talking to organizators to gain an understanding of the problem.

Step 03

User journey

To understand how the target group might interact with the web app in different situations we created user personas and user scenarios that were based on interviews.

Step 05

Prototyping

Starting from sketches we worked with prototyping and user tests till we created high visual fidelity prototypes with Adobe Xd that.

Step 02

Interviews

We talked with 5 people who have participated in the carnival and asked them to express their thoughts about the carnival and what they enjoyed and disliked.

Step 04

Sketches and wireframes

IA was key for easy, intuitive navigation and information placement. From first lines to wireframes we worked with card sorting and followed design patterns.

Step 06

Front-end development

After the final user test the development started. For this project, I learned how to create tab layout and trigger CSS with JavaScript to mimic application interactions.

Final prototype

We create a prototype for a web app that would provide Aalborg Karneval participants with useful information while partying. This was my first project using JavaScript and my focus was to create a layout that is powered by it and resembles a native application. For further development a database would be needed as the plan was to implement a function where users could create their own schedule and as well as connect with their friends and see their location on the map.
